# Random Forest Classification Project

A supervised machine learning classification project using tabular data.

## Project Overview

This project demonstrates a machine learning classification workflow using Python. The notebook includes data loading, data exploration, preprocessing, feature selection, train-test split, scaling, and model training.

The main models used in the notebook are:

- Decision Tree
- Random Forest

## How to Run

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/random-forest-classification-project.git
cd random-forest-classification-project
```

Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Open the notebook:

```bash
jupyter notebook notebooks/classification_model.ipynb
```
